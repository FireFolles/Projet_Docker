# Todo Backend

## Installation

```bash
cd Back
npm install
```

## Lancement en développement

```bash
npm run dev
```

Le serveur écoute par défaut sur http://localhost:3000

## Lancement en production
```bash
npm start
```

## Environnement attendu
- Crée un fichier `.env` dans `Back` avec ta config SUPABASE.

```
SUPABASE_URL=... (URL de ton instance supabase)
SUPABASE_ANON_KEY=... (clé anon ou service role)
```

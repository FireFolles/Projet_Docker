# Todo Frontend

## Installation (optionnelle)

Si vous souhaitez un serveur de dev local :

```bash
cd Front
npm install
```

## Lancement du serveur (mode dev ou prod)

```bash
npm run dev
```

Cela ouvre l'app sur http://localhost:3000 (backend) et http://localhost:5000 (frontend par défaut, selon le port affiché)

## Configuration
- Les appels API sont faits sur `http://localhost:3000/api/tasks`.
- Pas besoin d'adapter côté front si ce backend reste accessible sur ce port et cette adresse.
